This is the modified version of Kaldi toolkit for Bayesian adaptation approaches proposed by the published paper:
	Xurong Xie, Xunying Liu, Tan Lee, & Lan Wang (2021). Bayesian Learning for Deep Neural Network Adaptation. IEEE/ACM Transactions on Audio, Speech, and Language Processing.

The codes are based on nnet3, please see egs/swbd/s5c for running examples.
